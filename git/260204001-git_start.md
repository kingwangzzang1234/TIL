## git이란?

git은 Linus Torvalds 가 만든 분산형 버전관리 시스템.

전 세계 개발자 96%가 사용중(2022)

## git의 특징

- 분산형 저장소 지원
- 비선형적 개발


## 저장소 세팅하기

### README.md

README.md 는 저장소의 대문과 같은 역할을 하는 문서로 매우 중요함

### .gitignore

git이 추적하지 않도록 파일과 폴더에 추적 제한을 거는 파일

https://gitignore.io/ 를 참조하여 운영체제, 에디터, 언어, 프레임워크에 대한 설정
