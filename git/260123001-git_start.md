# Git

## git이란?

분산형 버전 관리 시스템

## 왜 git?

- 단순한 구조, 빠른 속도
- 분산형
- 비선형적 개발 가능

## git == github??

- git: VCS (System)
- github: Web Repository Service

# git command

```shell
$ git clone {repo addr}
$ git status
$ git add {filename}
$ git commit
$ git commit -m "{msg}"
$ git push origin main
$ git remote -v
```

- `git clone`: 원격 저장소를 로컬에 복제할 때 사용
